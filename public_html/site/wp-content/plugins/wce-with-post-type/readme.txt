== Description ==
This Add-on adds using post type for the WP CSV Exporter plugin.

== Installation ==
1. Upload the entire `wce-with-post-type` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. You must also activate 'WP CSV Exporter' through the 'Plugins' menu in WordPress.
